## 🚀 Como executar 

Este programa foi feito em **linguagem C++** e pode ser compilado em uma IDE como o **VS Code** (com compilador instalado).
No seu computador (VS Code ou outro editor):
1. Compile o código através do comando `g++ DesafioFinal.cpp stb_image.cpp gl_utils.cpp glfw3dll.a -I ".\include" -L "." -L ".\lib" -lOpenGL32 -lglew32 -lglfw3 -lm -o <nome do executável>`
2. Abra o executável gerado

> [!TIP]
> Para facilitar a compilação, foi criado o `run.bat`, para usá-lo, basta escolher o arquivo do projeto e o nome do executável que será gerado, dessa forma (estando na pasta raiz do projeto): `.\run.bat .\DesafioFinal.cpp <nome do executável>`.\
Ao rodar esse comando, a compilação será feita e o executável será aberto automaticamente.

## 💡 Sobre o Programa

Neste jogo, você está com fome e precisa sair de casa para buscar algo para comer. Mas não será tão fácil!

Existe um hamburguer do outro lado, mas não será simples chegar até ele.

No trajeto, você deverá:
- Evitar águas profundas, já que você não sabe nadar
- Desviar de orbes misteriosos e perigosos que rondam o caminho até seu almoço
- Evitar solos frágeis que colapsam e revelam buracos profundos. Cuidado para não cair neles!

Após chegar ao hamburguer e saciar a sua fome, você deverá voltar para casa para descansar. Somente então, você será vitorioso!
